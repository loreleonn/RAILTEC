# RAILTEC – Validaciones Automáticas ADIF

Mini‑app desarrollada en Python para la validación automática de parámetros de diseño ferroviario conforme a normativa ADIF.

## Objetivo
Automatizar la comprobación normativa en:
- Señalización
- Balizamiento
- Geometría del trazado ferroviario

mediante un motor de reglas programado.

## Tecnologías
- Python 3
- Control de versiones con Git

## Estado del proyecto
Prototipo funcional en desarrollo para el proyecto transdisciplinar RAILTEC.
